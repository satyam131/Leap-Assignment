# Leap-Assignment
A WebSocket-based Chat Application built with Spring Boot and React for real-time communication

#WebSocket Chat App
This repository contains the source code for a WebSocket-based chat application developed using Spring Boot on the backend and React on the frontend. The application enables real-time communication between users through a simple and intuitive interface. Explore the code to understand the implementation details and dependencies. Follow the setup instructions below to run the application locally and start chatting!

#Setup Instructions:

##Backend (Spring Boot):
Navigate to the backend directory.
Run ./mvnw spring-boot:run (or mvnw.cmd on Windows) to start the Spring Boot backend server.

##Frontend (React):
Navigate to the frontend directory.
Run npm install to install the necessary dependencies.
Run npm start to start the React frontend development server.

##Access the Application:
Open your browser and go to http://localhost:3000 to access the WebSocket Chat App.
Enjoy real-time communication with other users!
